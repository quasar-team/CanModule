Quickstart¶

This is how to quick start CANX (if you know already what this is):

  • download CANX and needed libs from nexus_can for your target platform
  • create a configuration file for CANX “to “play”. Download some examples from jenkins_can (Expand All->View->Config->*.xml)
  • run it (on your target platform): CANX-tester -T -cfg ./myconfig.xml enjoy the output
